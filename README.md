# My Area Problems 🗺️
**Agentic AI civic reporting platform — SDG 6 & SDG 11**

Citizens report local issues (water, roads, garbage) via photo + voice.
AI agents classify, retrieve relevant laws via RAG, and generate legal complaint letters addressed to the correct authority.

## Tech Stack (100% Free)
| Layer | Tool |
|-------|------|
| LLM | Groq API — Llama 3.1 8B |
| Agents | LangChain |
| Vector DB | ChromaDB (local) |
| Embeddings | sentence-transformers |
| Vision | BLIP-2 (HuggingFace) |
| Voice | Whisper (local) |
| Maps | Folium + OpenStreetMap |
| UI | Gradio |
| Deployment | HuggingFace Spaces |
